# 薪资计算器

一个免登录、开箱即用的欠薪、未续签双倍工资、社保与公积金补缴测算工具。数据默认只保存在用户当前浏览器中，不上传业务数据。

[在线使用](https://kun8964.github.io/Wage-Arrears-Calculator/)

## 功能

- 根据入职日期和统计截止日期，按月生成补缴明细并统计月数
- 根据开始欠薪月份生成工资状态，支持首个欠薪月按已发比例计算
- 分别统计社保、公积金公司实际缴纳月份与金额，再按月计算尚欠差额
- 支持设置公司实缴起止月份，期间内少缴金额会先扣除，期间外按未缴计算
- 根据合同期满日自动判断期满后持续用工满 1 个月并启用双倍工资（最多 11 个月）
- 提供独立、可关闭的工伤典型情形与申报期限初筛，不把尚未经认定和鉴定的待遇金额并入合计
- 导出报告末页提供全国通用的监察投诉、劳动仲裁、支付令、社保与公积金专项处理、律师及法律援助路径提示
- 采用黑、白与酸性绿构成的编辑式科技界面，支持桌面、平板和手机端响应式使用
- 本机保存、新建测算、JSON 备份与恢复
- CSV 明细导出
- 手机与桌面端自适应

## 本地运行

需要 Node.js `>=22.13.0`。

```bash
npm install
npm run dev
```

打开终端输出的本地地址即可使用。

## 构建与检查

```bash
npm run build
npm run lint
npm run typecheck
npm test
npm run test:e2e
```

构建产物位于 `dist/`。项目基于 Next.js、React、Tailwind CSS 与 vinext，可部署到兼容 Cloudflare Workers 的环境。

## 数据与隐私

- 页面不要求注册或登录。
- 保存的数据写入浏览器 `localStorage`。
- 清理浏览器数据会同时删除本机测算，重要案例请使用“备份”导出 JSON 文件。
- JSON 文件可在另一台设备或另一浏览器中通过“导入”恢复。

## 计算口径提示

本工具只用于初步核对。工资基数、社保和公积金比例、双倍工资起止日期、工伤认定及例外情形可能因地区和具体案件不同，正式主张前请核对当地现行规定或咨询专业人士。

社保和公积金公式中的“实际已缴金额”与“比例”必须采用相同口径：只填写单位缴费额时使用单位比例；填写单位与个人合计时使用双方比例之和。
